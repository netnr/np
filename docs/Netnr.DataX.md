# Netnr.DataX
数据转换

### 转换步骤
1. 配置原数据库连接、新数据库连接
2. 生成原表名和新表名的映射、原表列名和新表列名的映射
3. 手动调整表名、列名的映射
4. 开始表数据转换
	- 设置原表名
	- 设置原表数据查询 SQL
	- 设置新表数据清空 SQL
	- 根据表列名映射将原表填充到新表
	- 手动处理未映射的列数据
	- 表复制写入新库

### 特点
- 支持 MySQL、SQLite、Oracle、PostgreSQL、SQLServer
- 自动模糊匹配两库的表名、列名生成映射，仅改一小部分未能映射的项
- 动态编译运行，复制 cs 文件模板快速构建表数据转换
- 详细的日志输出
